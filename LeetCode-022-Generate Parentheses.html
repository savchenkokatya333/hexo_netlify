发表于 2018-12-29 | 更新于 2019-01-03 | 分类于 LeetCode日记

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

思路分析：

  • 对于要列出所有组合的题目，基本上可以首先考虑递归法，首先用left和right两个变量表示现有左右括号的数目，结果一定是left==3, right==3，所以一直递归就好了，当right == 3时将生成的字符串添加到结果中就好，注意递归的条件:left<3, right<left，因为一旦right>left，字符串一定是不合法的，会出现)(的情况，具体见代码。

理解递归时容易晕，有个图可能会好点，递归其实就是一种深度遍历，即首先将一条路走到黑，在本题中就是先一直加左括号，当left==3时再加右括号，所以可以看到结果出来的顺序，不断的走到黑，返回上一级，继续选一条路走到黑。

网上说这道题是一个回溯法的经典问题，回溯和递归有什么区别？回溯是一个算法思想，而递归是一种算法结构，在代码里递归就是自己调用自己，而回溯思想一般会用递归来实现，具体看参考链接吧。
